# 🎙️ Assistente Virtual por Voz (Python)

Este projeto implementa um **assistente virtual simples** usando **Processamento de Linguagem Natural (PLN)**, capaz de reconhecer comandos de voz, responder perguntas e executar ações como:

- **Dizer as horas** 🕒
- **Pesquisar na Wikipédia** 📚
- **Tocar músicas no YouTube** 🎵
- **Responder a comandos básicos** 💬

---

## 📌 Funcionalidades

- 🎤 **Reconhecimento de voz** (Speech-to-Text) usando `SpeechRecognition`
- 🗣️ **Síntese de voz** (Text-to-Speech) usando `pyttsx3`
- 🌐 Pesquisa automática na **Wikipédia**
- 📺 Reprodução de vídeos no **YouTube**
- 🖥️ Respostas faladas e exibidas no terminal
- ❌ Comando de **encerramento por voz**
